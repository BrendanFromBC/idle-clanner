# Idle Clanner

A companion web app for the idle RPG [Idle Clans](https://www.idleclans.com/) — a planning assistant, gear guide, and gold
optimizer, personalized to your specific accounts and skill levels.

**Core idea**: most players run 3 accounts simultaneously (one main, two alts focused on
gathering/crafting). Every feature here is built around a *team* of accounts, not a single
player — that's the main thing this tool does differently from other Idle Clans companion
sites.

This is a fan-made project, not affiliated with or endorsed by the Idle Clans developers.

## Features

- **Team dashboard** — skill levels and equipped gear for up to 3 accounts at once
- **Market browser** — live buy/sell prices, 1d/7d/30d averages, and a 24h price-history chart per item
- **Gear guide** — role-aware upgrade recommendations (combat BiS for your main, tool tiers for gatherer alts), with live market cost and "how to get it" info
- **Gold/hr optimizer** — ranks every unlocked activity by real profit/hr (market cost of inputs included), with a separate Ironman mode that values items at shop price and accounts for self-gathering time
- **Loot odds & planner** — monster drop tables grouped by in-game area, reverse "which monster drops this item" lookup, a melee kills/hr estimate from your actual equipped gear, and a cross-account wishlist

## Tech stack

Vite + React 18 + TypeScript, Tailwind CSS v4, TanStack Query, Zustand (persisted to
`localStorage`), React Router. No backend — everything runs client-side against the public
[Idle Clans API](https://query.idleclans.com) plus a static game database generated from
that same API.

## Getting started

```bash
npm install
npm run dev
```

## Data sources

- **Live**: the public Idle Clans API (`query.idleclans.com`) for player profiles and market prices. No API key required.
- **Generated**: `src/data/*.ts` files are built from the API's own `Configuration/game-data`
  dump via the scripts in `scripts/` (e.g. `npm run` isn't wired up for these yet — run
  `node scripts/generate-items.mjs` etc. directly). Re-run after a game update.
- **Curated**: `src/data/gear.ts` — which items count as "BiS" is hand-picked from the wiki, since that's a judgment call the API can't make.
- **Icons**: `src/data/icons.ts` is generated by resolving item/monster names against the
  [Idle Clans wiki](https://idleclans.wiki)'s MediaWiki API, since the official game API has
  no icon data at all. Icons are hotlinked directly from the wiki at runtime (not bundled in
  this repo) — please be mindful of that if you fork this and expect meaningful traffic.
